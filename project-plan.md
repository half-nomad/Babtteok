# 밥떡한끼 브랜드 슬라이드 프로젝트 계획서

## 📌 프로젝트 개요
**프로젝트명**: 밥떡한끼 (Babtteok) 브랜드 슬라이드 프레젠테이션  
**목적**: 30-50대 예비창업주 대상 밥버거-떡볶이 세트 프랜차이즈 사업 제안  
**기간**: 2025년 6월 29일 ~ 2025년 7월 19일 (3주)  
**개발팀**: 슬라이드 프로덕션 팀 (바닐라 JS 하이브리드 버전)

## 🎯 핵심 목표
1. **전환율 향상**: 사업설명회 참석 또는 상담 신청 15% 이상
2. **신뢰도 구축**: 구체적 수치와 현실적 분석으로 신뢰감 조성
3. **차별화**: 검증된 메뉴 조합 + 시장 공백 활용의 독특함 강조
4. **사용자 경험**: 모바일 최적화 및 인터랙티브 요소로 몰입도 증대

## 📊 타겟 분석
- **주요 타겟**: 30-50대 예비창업주
- **페인 포인트**: 높은 가맹비, 포화 시장, 불안한 수익성
- **해결 방안**: 8,000만원 합리적 투자, 검증된 메뉴, 시장 공백 활용
- **기대 효과**: 2년 내 투자 회수, 안정적 수익 창출

## 🎨 디자인 방향성
- **컬러**: 네이비(신뢰) + 그레이(안정) + 오렌지(포인트)
- **폰트**: 명조/고딕 혼합 (전문성 + 가독성)
- **톤앤매너**: 감성보다 구체적 수치와 현실적 분석 중심
- **레이아웃**: 정보 전달 우선, 차트와 표 중심의 논리적 구성

## 📋 슬라이드 구성 (6개 슬라이드)

### 슬라이드 1: 문제 정의/현황 분석
**핵심 메시지**: 소자본 창업시장의 현실과 한계  
**필요 요소**:
- 창업 실패율 통계 그래프
- 프랜차이즈 포화도 현황
- 임대료 vs 매출 비교 차트
- 창업주 고민 키워드 클라우드

### 슬라이드 2: 솔루션/아이디어 제시
**핵심 메시지**: 밥떡이의 차별화된 사업모델  
**필요 요소**:
- 초기 투자금 구성 파이 차트
- 손익분기점 시뮬레이션
- 하이브리드 운영 모델 다이어그램
- **인터랙티브**: ROI 계산기

### 슬라이드 3: 타겟/시장 분석
**핵심 메시지**: 성장하는 시장과 확실한 수요  
**필요 요소**:
- 1인 가구 증가 트렌드 그래프
- 배달 시장 성장률 데이터
- 떡볶이 프랜차이즈 매출 현황
- 지역별 창업 적합도 히트맵

### 슬라이드 4: 실행 계획/프로세스
**핵심 메시지**: 체계적 3단계 사업 확장  
**필요 요소**:
- 3단계 확장 로드맵 타임라인
- 가맹 조건 비교표
- 본부 지원 시스템 프로세스
- 성공 점포 사례 스터디

### 슬라이드 5: 결과/성과 예측
**핵심 메시지**: 안정적 투자 회수와 수익 창출  
**필요 요소**:
- 월별 손익 시뮬레이션 표
- 투자 회수 기간 그래프
- 브랜드 성장에 따른 점포 가치 상승 차트
- 리스크 대비 수익률 분석

### 슬라이드 6: 제안/Call to Action
**핵심 메시지**: 얼리어답터 모집 및 특별 혜택  
**필요 요소**:
- 창업주 성공 스토리 일러스트
- 얼리어답터 특별 혜택 정리
- 창업 상담 신청 버튼 (인터랙티브)
- QR 코드 (사업계획서 다운로드)

## 🛠️ 기술 스택

### 핵심 기술
- **HTML5**: 시맨틱 마크업
- **CSS3**: 플렉스박스, 그리드, 애니메이션
- **바닐라 JavaScript**: 핵심 로직 및 인터랙션

### 선택적 라이브러리
- **Chart.js**: 데이터 시각화 (차트, 그래프)
- **GSAP**: 고급 애니메이션 효과
- **Swiper.js**: 슬라이드 네비게이션
- **AOS**: 스크롤 애니메이션

### 개발 도구
- **Text Editor**: 코드 작성 및 편집
- **Desktop Commander**: 파일 관리 및 Git
- **Playwright**: 크로스브라우저 테스트
